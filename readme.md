This is one of two repositories created for the Jarwis AI website <a href="https://www.jarwis.ai/"><samp>jarwis.ai</samp></a>. If you have any questions or comments, feel free to contact us at <samp><a href="mailto:contact@jarwis.ai">contact@jarwis.ai</a></samp>.

<br>

To setup a dev environment:

```bash
# Clone the repository

# Install dependencies 
npm i

# Run the local dev server
npm run dev
```

To serve a production build:

```bash
# Install dependencies if not already done - 'npi i'

# Build for production
npm run build

# Serve the build using express
npm start
```
